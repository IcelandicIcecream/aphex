# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the main folder](https://github.com/changesets/changesets)

The way changesets work is that when you want to make a release, you run `pnpm changeset` and answer
the prompts. This will create a new changeset file in this folder describing the changes you've made.
When you're ready to release, run `pnpm changeset version` to consume the changesets and bump versions.
Then run `pnpm release` to build and publish.
